HandbrakeProcess

Defined in lib/handbrake.js:28

Runs HandbrakeCLI with the supplied options.

Parameters:

  • options Object | Thing | Array

    Options to pass directly to HandbrakeCLI

  • [onComplete] Function optional

    If passed, onComplete(stdout, stderr) will be called on completion.

Returns:

HandbrakeProcess: A HandbrakeProcess instance on which you can listen for events.

Example:

There are two ways to invoke run(), the first enables you to catch events as they happen:

var handbrake = require("handbrake-js");

var options = {
    input: "Eight Miles High.mov",
    output: "Eight Miles High.m4v",
    preset: "Normal"
};

handbrake.run(options)
    .on("output", console.log);
    .on("progress", function(encode){
        console.log(encode.percentComplete);
    })
    .on("complete", function(){ 
        console.log("Encode complete"); 
    });

The second method is to pass an onComplete callback. It's more convenient for short duration tasks: 

handbrake.run({ preset-list: true }, function(stdout, stderr){
    console.log(stdout);
});
